Embed content

Below are guidelines for embedding content on a page in a course. Content can be embeded that is hosted in the LMS course files or a public web site.

Use "Object" for content hosted in the LMS

Content that is not hosted on another website, but instead is stored as a file in the course can be embeded on the page using the "object" element.



Use "iFrame" for content hosted on another website

Content that is hosted on another website (i.e google drive) can be embedded using an iFrame.
