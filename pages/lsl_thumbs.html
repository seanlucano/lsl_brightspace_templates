Links

The links below are used for linking to other websites or for a file that you can download. Use an "ol" (ordered list) element to create a numbered list of links. Use "ul" (unordered list) for a bulleted lists of links.

  1. Another website
  2. Download this file
                     
<ol> 
   <li> <a href="#" target="_blank">Another website</a> </li> 
   <li> <a href="#">Download this file</a> </li> 
</ol>

Include a class of "new-window" to add the new window icon after the link. use the class "download" to add a download link icon. The addition of these icons is not required, but you may sometimes find it helpful to be abel to use to help students understand how a link will behave before they click on it.

  • Another website
  • Download this file
                     
<ul> 
   <li> <a class="new-window" href="#">Another website</a> </li> 
   <li> <a class="download" href="#">Download this file</a> </li> 
</ul>
